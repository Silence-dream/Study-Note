1、JavaScript数据类型。



![img](img/413)

```
JavaScript有7钟基本类型，1种引用类型。
引用数据类型（Object）又分为这几种常见的类型：
Array - 数组对象、RegExp - 正则对象、Date - 日期对象、Math - 数学函数、Function - 函数对象。


基础类型存储在栈内存，被引用或拷贝时，会创建一个完全相等的变量；

引用类型存储在堆内存，存储的是地址，多个引用指向同一个地址，这里会涉及一个“共享”的概念。
```

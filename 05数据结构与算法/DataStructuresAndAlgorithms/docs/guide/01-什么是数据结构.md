## 什么是数据结构

一个简单的理解比如定义了一个变量，然后赋值，然后再输出，这个过程就是数据结构。

```js
let a;
a = 10;
let array = [1, 2, 3, 4];
let person = {
  name: "张三",
  age: "18",
};
```

还不理解?用人话来说就是给一堆水果,然后进行分类,思考分类的过程就是算法,分类完成的结果就是数据结构。

## 常见的数据结构

- 数组(Array)
- 栈(Stack)
- 链表(Linked List)
- 队列(Queue)
- 堆(Heap)
- 树(Tree)
- 图(Graph)
- 哈希表(Hash)

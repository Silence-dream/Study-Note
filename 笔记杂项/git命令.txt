git配置用户名和邮箱
git config --global user.name "yourname"
git config --global user.email youreamil

git命令提交三部曲
git add .
git commit -m "注释"
git push 

git ssh秘钥生成
ssh-keygen -t rsa -C "youreamil"


git status  //显示工作目录和暂存区的状态

删除最后一次提交
命令很简单，但是不能马虎，不能多回滚或者少回滚，不然就麻烦不断了。

主要命令：

第一步：回滚上一次提交
git reset --hard HEAD^

就很简单，回滚master分支的上一次提交。

第二步：强制提交本地代码
git push origin master -f

让git识别大小写文件夹
git config core.ignorecase false
